Allgemein

Die Menge der ganzen Zahlen, bezeichnet mit dem Symbol ℤ, umfasst alle Zahlen ohne Dezimalstellen: also sowohl positive als auch negative ganze Zahlen sowie die Null.

ℤ = { ..., -3, -2, -1, 0, 1, 2, 3, ... }

Diese Zahlenmenge ist unendlich und wird in vielen Bereichen des Alltags und der Mathematik verwendet.

  • Abgeschlossenheit: Addition, Subtraktion und Multiplikation ergeben wieder ganze Zahlen.
  • Keine Brüche oder Kommazahlen: Werte wie 0.5 oder ⅓ sind nicht Teil der Menge ℤ.
  • Gegenzahlen: Jede ganze Zahl hat eine Gegenzahl. Z. B. ist die Gegenzahl von 7 gleich -7.
  • Die Null: Sie ist weder positiv noch negativ und wirkt in der Addition neutral.

Variabeln

In der Algebra werden Zahlen oft durch Buchstaben ersetzt. Diese Buchstaben nennt man Variablen. Sie stehen für unbekannte oder veränderliche Werte. Beispiele: a, b, c, x, y, z. Auch Potenzen wie a² oder x³ zählen dazu.

Grundrechenarten mit Variablen

  • x + x = 2x
  • 2x - x = x
  • x · x = x²
  • x ÷ x = 1 (vorausgesetzt x ≠ 0)

Binomische Formeln

Die drei binomischen Formeln lauten:

  1. (a + b)² = a² + 2ab + b²
  2. (a - b)² = a² - 2ab + b²
  3. (a + b)(a - b) = a² - b²

Beispiele

  • (x + 5)² = x² + 10x + 25
  • (x - 3)² = x² - 6x + 9
  • (x + 4)(x - 4) = x² - 16

Faktorisieren

Um einen Ausdruck in ein Produkt binomischer Formeln zu überführen („faktorisieren“), erkennt man zunächst das Muster:

  • Ist der Term a² + 2ab + b², so ist er (a + b)².
  • Ist der Term a² - 2ab + b², so ist er (a - b)².
  • Ist der Term a² - b², so ist er (a + b)(a - b).

Tipps

  • Achte auf das **Vorzeichen** des mittleren Terms (2ab) – es bestimmt + oder - in (a ± b)².
  • Ein **perfektes Quadrat** besteht immer aus drei Termen: a², 2ab, b².
  • Bei einer **Differenz** a² - b² einfach die Summanden addieren/subtrahieren: (a + b)(a - b).
  • Um eine Summe in eine binomische Formel zu verwandeln, versuche, **a** und **b** so zu wählen, dass die Koeffizienten passen.
