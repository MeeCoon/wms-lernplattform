1. Allgemein

Die Menge der ganzen Zahlen, bezeichnet mit dem Symbol ℤ, umfasst alle Zahlen ohne Dezimalstellen: also sowohl positive als auch negative ganze Zahlen sowie die Null.

ℤ = { ..., -3, -2, -1, 0, 1, 2, 3, ... }

Diese Zahlenmenge ist unendlich und wird in vielen Bereichen des Alltags und der Mathematik verwendet.

  • Abgeschlossenheit: Add./Subtr./Multiplikation liefern wieder ganze Zahlen.
  • Keine Dezimalwerte: 0.5 oder ⅓ gehören nicht zu ℤ.
  • Gegenzahlen: Zu jeder a ∈ ℤ gehört -a, mit a + (-a) = 0.
  • Null: neutrales Element der Addition.

Zahlen wie -5, 0 oder 14 sind ganze Zahlen, 3.14 oder -0.5 jedoch nicht.

Beispiele im Alltag: Temperatur (-4 °C), Kontostände (-20 €), Höhenangaben (–50 m).

2. Variabeln

Variablen (z. B. a, b, x, y) ersetzen Zahlen und machen Ausdrücke allgemein.

  • Addition: x + x = 2x – Wichtig: Basis gleich lassen und Koeffizienten addieren.
  • Subtraktion: 2x – x = x – Wichtig: Basis gleich lassen und Koeffizienten subtrahieren.

Multiplikation von Potenzen:
Wenn du zwei Potenzen derselben Basis multiplizierst, addierst du ihre Exponenten:


    xa × xb = xa+b
    Beispiel: x2 × x3 = x5

Division von Potenzen:
Beim Teilen zweier Potenzen derselben Basis subtrahierst du die Exponenten:


    xa ÷ xb = xa−b
    Beispiel: x2 ÷ x = x1 = x

WICHTIG: Jede Variable x ohne sichtbaren Exponenten entspricht x1. Die Basis muss stets identisch sein; zwei verschiedene Variablen (z. B. x und y) lassen sich nicht direkt addieren oder subtrahieren.

3. Binomische Formeln

Binomische Formeln ermöglichen das schnelle Erweitern und Faktorisieren von Ausdrücken der Form (a ± b)² und (a + b)(a - b).

3.1 Herleitung von (a + b)²

Schreibe (a + b)² als (a + b)(a + b) und verwende FOIL:

  • a·a = a²
  • a·b = ab
  • b·a = ba = ab
  • b·b = b²

Daher: (a + b)² = a² + ab + ab + b² = a² + 2ab + b².

3.2 Herleitung von (a - b)²

Analog: (a - b)² = (a - b)(a - b):

  • a·a = a²
  • a·(-b) = -ab
  • -b·a = -ba = -ab
  • -b·(-b) = b²

Daher: (a - b)² = a² - ab - ab + b² = a² - 2ab + b².

3.3 Herleitung von (a + b)(a - b)

Multipliziere (a + b)(a - b):

  • a·a = a²
  • a·(-b) = -ab
  • b·a = ba = ab
  • b·(-b) = -b²

Daher: (a + b)(a - b) = a² - ab + ab - b² = a² - b².

3.4 Beispiele

  • (x + 5)² = x² + 10x + 25
  • (x - 3)² = x² - 6x + 9
  • (x + 4)(x - 4) = x² - 16

3.5 Faktorisieren

Erkenne das Muster:

  • a² + 2ab + b² → (a + b)²
  • a² - 2ab + b² → (a - b)²
  • a² - b² → (a + b)(a - b)

3.6 Tipps

  • Achte auf das Vorzeichen des mittleren Terms 2ab.
  • Suche bei Faktorisierung stets nach perfekten Quadraten oder Differenzen.
  • Teste beim Umformen verschiedener a- und b-Werte, ob der Koeffizient passt.
